# ALPS Research Conference

Standalone Astro project for the ALPS Research Conference landing page.

## Commands

```bash
bun install
bun run dev
bun run build
```

The production-ready static site is generated in `dist/`.

Set `SITE_URL` during builds if you want Astro to emit absolute metadata URLs:

```bash
SITE_URL=https://your-domain.com bun run build
```
